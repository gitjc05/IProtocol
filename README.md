# IProtocol
- This is a simple light weight multi-layer, cross platform, modular protocol interface for udp, tcp, and serial port communication.
- The purpose of this library is to create light weight networks for SOMs, embedded systems, edge devices, and host desktops to communicate.
- WSLGuide shows a step by step guide for linux debugging with visual studio.
- This uses the c++ [ASIO-standalone][https://think-async.com/Asio/index.html] library, but will be slimmed down to raw windows and posix api when I get the time to do so.
